# 恋爱大师文件读取功能说明 📄

## 🎯 功能概述

恋爱大师现在支持读取用户上传的文本文件，可以分析聊天记录、日记、信件等文本内容，提供更精准的恋爱建议。

## ✅ 已添加的工具

在 `LoveAppToolRegistration.java` 中添加了 `FileOperationTool`：

```java
new FileOperationTool() // 支持读取用户上传的文本
```

## 📝 支持的文件格式

### 当前 FileOperationTool 支持的格式

`FileOperationTool` 使用 `FileUtil.readUtf8String()`，可以读取**所有文本格式**的文件：

#### ✅ 文本文件

- `.txt` - 纯文本
- `.log` - 日志文件
- `.csv` - CSV 数据文件
- `.md` - Markdown 文档
- `.json` - JSON 数据
- `.xml` - XML 文件
- `.yaml` / `.yml` - YAML 配置
- `.html` - HTML 文件
- `.css` - CSS 文件
- `.js` - JavaScript 代码
- `.java` - Java 代码
- `.py` - Python 代码
- `.sql` - SQL 脚本
- `.conf` / `.ini` / `.properties` - 配置文件

#### ❌ 二进制文件（不支持）

- `.docx` - Word 文档
- `.pdf` - PDF 文档
- `.xlsx` - Excel 表格
- `.jpg` / `.png` - 图片文件
- `.mp3` / `.mp4` - 音视频文件
- `.zip` / `.rar` - 压缩文件

### 扩展支持：DocumentReaderTool

如果需要读取 Word、PDF 等格式，可以使用 `DocumentReaderTool`（已创建，需要额外配置）：

#### 支持的格式

- ✅ `.docx` - Word 文档（使用 Apache POI）
- ✅ `.pdf` - PDF 文档（使用 Apache PDFBox）
- ✅ 所有文本格式（自动检测）

#### 配置步骤

1. **添加依赖到 pom.xml**：

```xml
<!-- Apache POI for Word documents -->
<dependency>
    <groupId>org.apache.poi</groupId>
    <artifactId>poi-ooxml</artifactId>
    <version>5.2.3</version>
</dependency>

<!-- Apache PDFBox for PDF documents -->
<dependency>
    <groupId>org.apache.pdfbox</groupId>
    <artifactId>pdfbox</artifactId>
    <version>2.0.29</version>
</dependency>
```

2. **在 LoveAppToolRegistration 中注册**：

```java
ToolCallback[] localTools = ToolCallbacks.from(
    new TavilyWebSearchTool(tavilySearchApiKey),
    new MailSendTool(),
    new DateLocationTool(),
    new FileOperationTool(),      // 文本文件
    new DocumentReaderTool()      // Word、PDF 文档
);
```

3. **使用示例**：

```
用户：我把聊天记录保存在 chat.docx 里了，你能帮我分析吗？

AI：[自动调用 readDocument("chat.docx")]
    好的，我已经读取了你的 Word 文档...
```

## 📝 FileOperationTool 功能

### 1. 读取文件 (readFile)

- **功能**：读取指定文件的内容
- **参数**：`fileName` - 文件名
- **返回**：文件的文本内容（UTF-8编码）
- **存储位置**：`{FILE_SAVE_DIR}/file/`

### 2. 写入文件 (writeFile)

- **功能**：将内容写入文件
- **参数**：
  - `fileName` - 文件名
  - `content` - 要写入的内容
- **返回**：写入成功的消息
- **存储位置**：`{FILE_SAVE_DIR}/file/`

## 🔧 技术实现

### 文件存储路径

```java
private final String FILE_DIR = FileConstant.FILE_SAVE_DIR + "/file";
```

### 读取文件实现

```java
@Tool(description = "Read content from a file")
public String readFile(@ToolParam(description = "Name of the file to read") String fileName) {
    String filePath = FILE_DIR + "/" + fileName;
    try {
        return FileUtil.readUtf8String(filePath);
    } catch (Exception e) {
        return "Error reading file: " + e.getMessage();
    }
}
```

### 写入文件实现

```java
@Tool(description = "Write content to a file")
public String writeFile(
    @ToolParam(description = "Name of the file to write") String fileName,
    @ToolParam(description = "Content to write to the file") String content) {
    String filePath = FILE_DIR + "/" + fileName;
    try {
        FileUtil.mkdir(FILE_DIR);
        FileUtil.writeUtf8String(content, filePath);
        return "File written successfully to: " + filePath;
    } catch (Exception e) {
        return "Error writing to file: " + e.getMessage();
    }
}
```

## 📋 前端集成建议

### 1. 添加文件上传功能

在 `LoveAppView.vue` 中添加文件上传组件：

```vue
<template>
  <!-- 在输入框上方添加文件上传区域 -->
  <div class="file-upload-section">
    <input
      type="file"
      ref="fileInput"
      @change="handleFileUpload"
      accept=".txt,.md,.log"
      style="display: none"
    />
    <button class="upload-btn" @click="$refs.fileInput.click()">
      <svg><!-- 上传图标 --></svg>
      上传文本文件
    </button>
    <span v-if="uploadedFile" class="file-name">
      {{ uploadedFile.name }}
    </span>
  </div>
</template>

<script setup>
const uploadedFile = ref(null);

const handleFileUpload = async (event) => {
  const file = event.target.files[0];
  if (!file) return;

  // 上传文件到服务器
  const formData = new FormData();
  formData.append("file", file);

  try {
    const response = await fetch("/api/upload", {
      method: "POST",
      body: formData,
    });

    if (response.ok) {
      uploadedFile.value = file;
      // 自动发送消息让 AI 分析文件
      inputMessage.value = `我上传了一个文件 ${file.name}，请帮我分析一下`;
      sendMessage();
    }
  } catch (error) {
    console.error("文件上传失败:", error);
  }
};
</script>
```

### 2. 添加文件上传 API

创建文件上传接口：

```java
@PostMapping("/upload")
public ResponseEntity<String> uploadFile(@RequestParam("file") MultipartFile file) {
    try {
        String fileName = file.getOriginalFilename();
        String filePath = FILE_DIR + "/" + fileName;

        // 保存文件
        FileUtil.mkdir(FILE_DIR);
        file.transferTo(new File(filePath));

        return ResponseEntity.ok("文件上传成功: " + fileName);
    } catch (Exception e) {
        return ResponseEntity.status(500).body("文件上传失败: " + e.getMessage());
    }
}
```

## 🔒 安全注意事项

### 1. 文件类型限制

建议只允许上传文本文件：

- `.txt` - 纯文本
- `.md` - Markdown
- `.log` - 日志文件
- `.csv` - CSV 数据

### 2. 文件大小限制

```java
// 限制文件大小为 5MB
if (file.getSize() > 5 * 1024 * 1024) {
    return "文件太大，请上传小于 5MB 的文件";
}
```

### 3. 文件名安全

```java
// 清理文件名，防止路径遍历攻击
String safeFileName = fileName.replaceAll("[^a-zA-Z0-9._-]", "_");
```

### 4. 隐私保护

- 用户上传的文件应该只能被该用户访问
- 建议使用 `chatId` 作为文件夹名称隔离不同用户的文件
- 定期清理过期文件

```java
@Tool(description = "List all files for current user")
public String listFiles(@ToolParam(description = "Chat ID") String chatId) {
    String userDir = FILE_DIR + "/" + chatId;
    File dir = new File(userDir);
    if (!dir.exists()) {
        return "No files found";
    }
    return String.join("\n", dir.list());
}
```

```java
@Tool(description = "Delete a file")

    @ToolParam(description = "Chat ID") String chatId,
```
