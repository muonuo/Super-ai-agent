# 智能体死循环检测改进方案

## 概述

本文档分析当前 `DeadLoopDetector` 的实现，并提供改进建议。

---

## 当前实现分析

`DeadLoopDetector` 位于 `src/main/java/com/monuo/superaiagent/agent/detector/DeadLoopDetector.java`，实现了以下几种检测方式：

| 检测类型 | 实现方式 | 阈值 |
|---------|---------|------|
| 完全重复 | 字符串相等比较 | `duplicateThreshold = 2` |
| 语义重复 | Jaccard 字符相似度 | `similarityThreshold = 0.8` |
| 工具重复 | 工具名+参数哈希比较 | 最近5次调用 |
| 连续失败 | 失败次数统计 | `maxConsecutiveFailures = 3` |

### 干预级别

```java
public enum InterventionLevel {
    NONE,      // 无需干预
    PROMPT,    // 级别1: 添加提示
    WARNING,   // 级别2: 强烈警告
    TERMINATE  // 级别3: 强制终止
}
```

---

## 存在的问题

### 1. Jaccard 相似度对中文效果不佳

**问题**: 当前按字符分割 (`s1.split("")`)，对中文来说粒度过细。

```java
// 当前实现
Set<String> set1 = new HashSet<>(Arrays.asList(s1.split("")));
```

例如：
- "你好" → {'你', '好'}
- "您好" → {'您', '好'}
- 相似度 = 1/3 ≈ 0.33，但实际含义很接近

**影响**: 可能导致误判，正常表达被误认为重复。

### 2. 缺少最大步数限制

**问题**: Agent 执行循环没有最大步数上限，可能无限循环。

**风险**: 如果检测逻辑有漏洞，可能导致程序卡死。

### 3. 工具调用历史无清理机制

**问题**: `toolCallHistory` 会无限增长。

```java
public void recordToolCall(String toolName, String arguments) {
    // 永远只添加，不删除
    toolCallHistory.add(record);
}
```

**影响**: 长时间运行会导致内存溢出。

### 4. 缺少振荡行为检测

**问题**: Agent 可能在两个状态之间来回切换（A→B→A→B），而不是完全重复。

**示例**:
```
调用 search(keyword="天气")
调用 search(keyword="温度")
调用 search(keyword="天气")  ← 振荡
调用 search(keyword="温度")  ← 振荡
```

### 5. 缺少时间窗口检测

**问题**: 短时间内的重复和长时间后的重复严重程度应该不同。

**示例**:
- 1分钟内重复5次 → 严重问题
- 1小时内重复5次 → 可能正常

## 改进方案

### 1. 改进中文语义相似度算法

#### 方案 A: 使用 n-gram 分词

```java
private double calculateNgramSimilarity(String s1, String s2, int n) {
    if (s1 == null || s2 == null) return 0.0;

    List<String> ngrams1 = getNgrams(s1, n);
    List<String> ngrams2 = getNgrams(s2, n);

    Set<String> set1 = new HashSet<>(ngrams1);
    Set<String> set2 = new HashSet<>(ngrams2);

    Set<String> intersection = new HashSet<>(set1);
    intersection.retainAll(set2);

    Set<String> union = new HashSet<>(set1);
    union.addAll(set2);

    return union.isEmpty() ? 0.0 : (double) intersection.size() / union.size();
}

private List<String> getNgrams(String text, int n) {
    List<String> ngrams = new ArrayList<>();
    for (int i = 0; i <= text.length() - n; i++) {
        ngrams.add(text.substring(i, i + n));
    }
    return ngrams;
}
```

**使用**: `calculateNgramSimilarity(response, recent, 2)` - 2-gram 对中文效果更好。

```java
private JiebaSegmenter segmenter = new JiebaSegmenter();

}
```

### 4. 添加振荡行为检测

```java
/**
 * 检查是否在两个工具调用之间振荡
 * 检测模式: A→B→A→B
 */
private boolean isOscillating() {
    if (toolCallHistory.size() < 4) {
        return false;
    }

    int n = toolCallHistory.size();
    // 检查最后2次和倒数第4次是否相同
    ToolCallRecord last = toolCallHistory.get(n - 1);
    ToolCallRecord secondLast = toolCallHistory.get(n - 2);
    ToolCallRecord fourthLast = toolCallHistory.get(n - 4);
    ToolCallRecord thirdLast = toolCallHistory.get(n - 3);

    // A→B→A→B 模式
    return last.getToolName().equals(thirdLast.getToolName())
        && secondLast.getToolName().equals(fourthLast.getToolName())
        && !last.getToolName().equals(secondLast.getToolName());
}

/**
 * 检测是否陷入循环
 */
public DetectionResult detect() {

    // ... 其他检测 ...

    // 检查振荡行为
    if (isOscillating()) {
        return new DetectionResult(
            InterventionLevel.WARNING,
            "检测到振荡行为模式(A→B→A→B)",
            "检测到你在两个操作之间来回切换，请尝试新的解决思路。"
        );
    }

    return new DetectionResult(InterventionLevel.NONE, "正常", null);
}
```

### 5. 添加时间窗口检测

```java
// 时间窗口配置 (毫秒)
private static final long TIME_WINDOW_MS = 60000; // 1分钟
private static final int MAX_CALLS_IN_WINDOW = 5;

/**
 * 检查时间窗口内的调用次数
 */
private boolean hasTooManyCallsInTimeWindow() {
    long now = System.currentTimeMillis();
    long recentCalls = toolCallHistory.stream()
        .filter(r -> now - r.getTimestamp() < TIME_WINDOW_MS)
        .count();
    return recentCalls > MAX_CALLS_IN_WINDOW;
```

## 完整改进代码

```java
package com.monuo.superaiagent.agent.detector;

import lombok.Data;
import lombok.extern.slf4j.Slf4j;

import java.util.*;
import java.util.stream.Collectors;

/**
 * 防死循环检测器 (改进版)
 */
@Data
@Slf4j
public class DeadLoopDetector {

    // ========== 可配置参数 ==========

    /** 重复响应阈值 */
    private int duplicateThreshold = 2;

    /** 最大连续失败次数 */
    private int maxConsecutiveFailures = 3;

    /** 语义相似度阈值 */
    private double similarityThreshold = 0.8;

    /** 最大执行步数 */
    private int maxSteps = 20;

    /** 最大记录最近响应数量 */
    private int maxRecentResponses = 5;

    /** 最大工具调用历史记录数 */
    private int maxToolCallHistory = 20;

    /** n-gram 分词大小 (中文建议2) */
    private int ngramSize = 2;

    // ========== 时间窗口配置 ==========
    private static final long TIME_WINDOW_MS = 60000; // 1分钟
    private static final int MAX_CALLS_IN_WINDOW = 5;

    // ========== 状态变量 ==========

    /** 当前连续失败计数 */
    private int consecutiveFailures = 0;

    /** 当前执行步数 */
    private int currentSteps = 0;

    /** 已尝试的工具集合 */
    private Set<String> attemptedTools = new HashSet<>();

    /** 最近响应列表 */
    private List<String> recentResponses = new ArrayList<>();

    /** 工具调用历史 */
    private List<ToolCallRecord> toolCallHistory = new ArrayList<>();

    /**
     * 工具调用记录
     */
    @Data
    public static class ToolCallRecord {
        private String toolName;
        private String argumentsHash;
        private long timestamp;
    }

    /**
     * 干预级别
     */
    public enum InterventionLevel {
        NONE,      // 无需干预
        PROMPT,    // 级别1: 添加提示
        WARNING,   // 级别2: 强烈警告
        TERMINATE  // 级别3: 强制终止
    }

    /**
     * 检测结果
     */
    @Data
    public static class DetectionResult {
        private InterventionLevel level;
        private String message;
        private String suggestedPrompt;

        public DetectionResult(InterventionLevel level, String message, String suggestedPrompt) {
            this.level = level;
            this.message = message;
            this.suggestedPrompt = suggestedPrompt;
        }
    }

    /**
     * 检测是否陷入循环
     */
    public DetectionResult detect() {
        // 1. 检查最大步数限制
        if (currentSteps >= maxSteps) {
            return new DetectionResult(
                InterventionLevel.TERMINATE,
                "达到最大步数限制(" + maxSteps + "步)",
                "你已尝试过多步骤，请直接给出最终答案或承认无法完成。"
            );
        }
        currentSteps++;

        // 2. 检查完全重复
        if (isRecentResponseDuplicate()) {
            return new DetectionResult(
                InterventionLevel.PROMPT,
                "检测到完全重复响应",
                getStuckPrompt()
            );
        }

        // 3. 检查语义重复
        if (!recentResponses.isEmpty()) {
            String lastResponse = recentResponses.get(recentResponses.size() - 1);
            if (isSemanticDuplicate(lastResponse)) {
                return new DetectionResult(
                    InterventionLevel.WARNING,
                    "检测到语义重复",
                    getWarningPrompt()
                );
            }
        }

        // 4. 检查工具重复调用
        if (hasTooManyRepeatedToolCalls()) {
            return new DetectionResult(
                InterventionLevel.WARNING,
                "检测到工具重复调用",
                getWarningPrompt()
            );
        }

        // 5. 检查振荡行为 (A→B→A→B)
        if (isOscillating()) {
            return new DetectionResult(
                InterventionLevel.WARNING,
                "检测到振荡行为模式",
                "检测到你在两个操作之间来回切换，请尝试新的解决思路。"
            );
        }

        // 6. 检查时间窗口内的频繁调用
        if (hasTooManyCallsInTimeWindow()) {
            return new DetectionResult(
                InterventionLevel.PROMPT,
                "检测到短时间内频繁调用",
                getWarningPrompt()
            );
        }

        // 7. 检查连续失败
        if (hasReachedMaxFailures()) {
            return new DetectionResult(
                InterventionLevel.TERMINATE,
                "达到最大连续失败次数(" + maxConsecutiveFailures + "次)",
                null
            );
        }

        return new DetectionResult(InterventionLevel.NONE, "正常", null);
    }

    /**
     * 记录响应
     */
    public void recordResponse(String response) {
        if (response != null && !response.isEmpty()) {
            if (recentResponses.size() >= maxRecentResponses) {
                recentResponses.remove(0);
            }
            recentResponses.add(response);
        }
    }

    /**
     * 记录工具调用
     */
    public void recordToolCall(String toolName, String arguments) {
        if (toolName == null) {
            return;
        }
        ToolCallRecord record = new ToolCallRecord();
        record.setToolName(toolName);
        record.setArgumentsHash(arguments != null ? String.valueOf(arguments.hashCode()) : "");
        record.setTimestamp(System.currentTimeMillis());

        // 容量限制
        if (toolCallHistory.size() >= maxToolCallHistory) {
            toolCallHistory.remove(0);
        }

        toolCallHistory.add(record);
        attemptedTools.add(toolName);
    }

    /**
     * 记录失败
     */
    public void recordFailure() {
        consecutiveFailures++;
    }

    /**
     * 重置状态
     */
    public void reset() {
        consecutiveFailures = 0;
        currentSteps = 0;
        attemptedTools.clear();
        recentResponses.clear();
        toolCallHistory.clear();
    }

    // ========== 私有检测方法 ==========

    /**
     * 检查最近响应是否完全重复
     */
    private boolean isRecentResponseDuplicate() {
        if (recentResponses.isEmpty()) {
            return false;
        }
        String lastResponse = recentResponses.get(recentResponses.size() - 1);
        if (lastResponse == null || lastResponse.isEmpty()) {
            return false;
        }
        long matchCount = recentResponses.stream()
                .filter(r -> r.equals(lastResponse))
                .count();
        return matchCount >= duplicateThreshold;
    }

    /**
     * 检查语义相似度 (使用 n-gram)
     */
    private boolean isSemanticDuplicate(String response) {
        if (recentResponses.isEmpty() || response == null) {
            return false;
        }
        for (String recent : recentResponses) {
            double similarity = calculateNgramSimilarity(response, recent);
            if (similarity >= similarityThreshold) {
                return true;
            }
        }
        return false;
    }

    /**
     * 计算 n-gram 相似度
     */
    private double calculateNgramSimilarity(String s1, String s2) {
        if (s1 == null || s2 == null || s1.isEmpty() || s2.isEmpty()) {
            return 0.0;
        }
        List<String> ngrams1 = getNgrams(s1, ngramSize);
        List<String> ngrams2 = getNgrams(s2, ngramSize);

        Set<String> set1 = new HashSet<>(ngrams1);
        Set<String> set2 = new HashSet<>(ngrams2);

        Set<String> intersection = new HashSet<>(set1);
        intersection.retainAll(set2);

        Set<String> union = new HashSet<>(set1);
        union.addAll(set2);

        return union.isEmpty() ? 0.0 : (double) intersection.size() / union.size();
    }

    /**
     * 获取 n-gram 列表
     */
    private List<String> getNgrams(String text, int n) {
        List<String> ngrams = new ArrayList<>();
        for (int i = 0; i <= text.length() - n; i++) {
            ngrams.add(text.substring(i, i + n));
        }
        return ngrams;
    }

    /**
     * 检查工具重复调用
     */
    private boolean hasTooManyRepeatedToolCalls() {
        if (toolCallHistory.size() < 3) {
            return false;
        }
        int recentCount = Math.min(toolCallHistory.size(), 5);
        for (int i = toolCallHistory.size() - recentCount; i < toolCallHistory.size() - 1; i++) {
            ToolCallRecord current = toolCallHistory.get(i);
            for (int j = i + 1; j < toolCallHistory.size(); j++) {
                ToolCallRecord next = toolCallHistory.get(j);
                if (current.getToolName().equals(next.getToolName())
                        && current.getArgumentsHash().equals(next.getArgumentsHash())) {
                    return true;
                }
            }
        }
        return false;
    }

    /**
     * 检查振荡行为 (A→B→A→B)
     */
    private boolean isOscillating() {
        if (toolCallHistory.size() < 4) {
            return false;
        }

        int n = toolCallHistory.size();
        ToolCallRecord last = toolCallHistory.get(n - 1);
        ToolCallRecord secondLast = toolCallHistory.get(n - 2);
        ToolCallRecord thirdLast = toolCallHistory.get(n - 3);
        ToolCallRecord fourthLast = toolCallHistory.get(n - 4);

        // A→B→A→B 模式
        return last.getToolName().equals(thirdLast.getToolName())
            && secondLast.getToolName().equals(fourthLast.getToolName())
            && !last.getToolName().equals(secondLast.getToolName());
    }

    /**
     * 检查时间窗口内的调用次数
     */
    private boolean hasTooManyCallsInTimeWindow() {
        if (toolCallHistory.size() < MAX_CALLS_IN_WINDOW) {
            return false;
        }
        long now = System.currentTimeMillis();
        long recentCalls = toolCallHistory.stream()
            .filter(r -> now - r.getTimestamp() < TIME_WINDOW_MS)
            .count();
        return recentCalls > MAX_CALLS_IN_WINDOW;
    }

    /**
     * 检查是否达到最大失败次数
     */
    private boolean hasReachedMaxFailures() {
        return consecutiveFailures >= maxConsecutiveFailures;
    }

    /**
     * 获取陷入循环的提示
     */
    private String getStuckPrompt() {
        return "检测到重复响应模式。你可能陷入了循环。请尝试以下策略：\n" +
                "1. 重新分析问题，可能需要不同的解决思路\n" +
                "2. 尝试使用不同的工具或方法\n" +
                "3. 如果任务确实无法完成，明确告知用户并终止\n" +
                "4. 避免重复已尝试过的无效路径";
    }

    /**
     * 获取强烈警告提示
     */
    private String getWarningPrompt() {
        return "【严重警告】检测到重复行为模式！\n" +
                "你正在重复相同的思考或行动，这可能导致任务无法完成。\n" +
                "请立即停止这种行为，并尝试以下策略：\n" +
                "1. 重新分析用户需求和当前状态\n" +
                "2. 考虑完全不同的解决思路\n" +
                "3. 如果当前方法不奏效，放弃并尝试新方法\n" +
                "4. 明确告知用户当前困境，不要重复无效尝试";
    }
}
```

## 测试建议

### 单元测试用例

```java
@Test
void testMaxStepsDetection() {
    DeadLoopDetector detector = new DeadLoopDetector();
    detector.setMaxSteps(3);

    // 执行3步
    for (int i = 0; i < 3; i++) {
        DetectionResult result = detector.detect();
        assertEquals(InterventionLevel.NONE, result.getLevel());
    }

    // 第4步应该触发终止
    DetectionResult result = detector.detect();
    assertEquals(InterventionLevel.TERMINATE, result.getLevel());
    assertTrue(result.getMessage().contains("最大步数"));
}

@Test
void testOscillationDetection() {

    detector.recordToolCall("toolA", "args1");
    detector.recordToolCall("toolB", "args1");

    DetectionResult result = detector.detect();
    assertEquals(InterventionLevel.WARNING, result.getLevel());
    assertTrue(result.getMessage().contains("振荡"));
}

    detector.recordResponse("今天天气不错");
```

## 总结

| 改进项 | 优先级 | 复杂度 |
|-------|--------|--------|
| 添加最大步数限制 | 高 | 低 |
| 资源清理机制 | 高 | 低 |
| n-gram 相似度 | 中 | 中 |
| 振荡行为检测 | 中 | 低 |
| 时间窗口检测 | 低 | 低 |
| 配置化 Builder | 低 | 中 |

整体来说，原有的死循环检测架构已经比较完善，改进主要集中在**稳定性**和**中文处理**方面。
